## Usage
Install the bot `trains` to a channel and use the following commands
 - /ttrains ... to ...
 - /tset x ... to ...
 - /tget x
 - /tdelete ...
 - /tinfo x
 - /tinfo all
where `x` is a string e.g. home/work/shops. This string cannot take any of the following commands above.
